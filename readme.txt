This plugin facilitates in assigning and displaying more than one author on a post.

== Description ==
On Admin side :

1.It provides a metabox labelled 'Contributor', which contains list of all authors on post-editor page.

2.User (Admin/Editor/Author) can tick one or more authors from the list by checking the checkbox.

On Front end :

1.It provides a contibutor box below the post content where contributor names with their Gravatars are shown.
 
2.The link to respective 'author' page will also be there.

== Installation ==
Upload the wpi-multiple-contributors folder to the /wp-content/plugins/ directory
Activate the Wpi Multiple Contributors plugin through the 'Plugins' screen in WordPress Dashboard.
